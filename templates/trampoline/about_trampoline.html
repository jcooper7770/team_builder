Recording turns

Type your skills into the text area which says "Input your routines here"

Some tips for recording your turns:

  • Input multiple skills by separating skill by spaces. i.e. 40o 40<
  • Use '%' or '--' to indicate the jumps were stopped but restarted the same turn
  • Input multiple turns at once by entering each turn on a separate line
  • Add a comment on a turn by adding a new line and starting the comment with "#" or '-' (See below example)
  • Utilize the shortucts (see next section below).
  • Copy a previously written turn by clicking on the number next to to the turn
  • Use the dropdowns to choose skills
  • Use the recommended skills which show up below the log input field by clicking on them.

Creating Routines

To save routines for a user from the logger page:

  • Complusory: Type "comp:routine skills". i.e. comp:40o 41o 40< 41<
  • Optional: Type "opt:routine skills". i.e. opt:12001< 811< 12001o 813<

Then to use them:

  • Compulsory: Type compulsory
  • Optional: Type optional

You can also save routines in the user profile by clicking on your username in the navigation bar at the top of the page.
